Proyecto: Timer diseñado para tecnica pomodoro de estudio usando tkinter.
Emilio Sanchez
tkinter
Sirve para crear una interfaz de usuario usando codigo en python.
La instale por medio de Pycharm, no use pip install. Esta libreria se puede instalar facilmente manualmente desde pycharm.
Para ejecutar el programa es necesario que descargues 2 librerias. La libreria de tkinter y la de math. La de tkinter es usada para crear la interfaz de usuario y la de math para la logica del segundero. Se requiere descargar la imagen "tomatones.png" y añadirla a la IDE para que el programa funcione. Finalmente, copiar el código de el archivo main.py y abrirlo en la IDE en el mismo directorio que la imagen y correr el programa.
Ejemplo de uso: ingresar la cantidad de tiempo en minutos que se desea trabajar por sesion, seguido del tiempo de descanso. Este timer cambiará entre el tiempo de trabajo y el tiempo de descanso.
